# p0insettia - iBoot payload
- Payload for patching kernel on iOS 10 iBoot  
- Base system is xerub's [iBex](https://github.com/xerub/ibex)

## Supported environments  
- iOS 10.0 or later (armv7)  

## Make 
```
./mk.sh
```
